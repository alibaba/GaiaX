
## bootstrap project
```
yarn bootstrap
```

## build project
```
yarn build
```

## run demo
```
cd packages/gaiax-taro-demo
yarn dev:h5
yarn dev:weapp
```

## 如何监听GaiaStudio

获取本机ip地址，赋值给GXFastPreview.studioIpAddress变量
